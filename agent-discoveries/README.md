# Agent Discoveries

This directory is a cache for AI agents to store their findings, architectural mappings, and other documentation that can speed up project exploration for future agent sessions.

## Guidelines for Agents
- Use descriptive filenames (e.g., `routing-overview.md`, `component-hierarchy.md`).
- Focus on high-level insights that are not immediately obvious from the file structure.
- Keep documentation up to date when significant architectural changes occur.
